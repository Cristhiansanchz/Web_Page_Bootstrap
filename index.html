    Características

    Multiproyecto

    Con Nayad Service Desk maneja múltiples proyectos desde la misma consola, usando una base de datos y un servidor único, de manera que los administradores del sistema pueden realizar la gestión de los diferentes procesos de tu organización.

    Autentificación

    La integración de los sistemas de autenticación de Nayad Service Desk con LDAP (directorio activo) ofrece al usuario un servicio de autenticación unificada seguro para el ingreso a la Mesa de Servicio.

    Web Service

    Utiliza una integración rápida y segura entre plataformas y programas diferentes, disminuyendo los tiempos de respuesta para tus usuarios.
